# 尝试
在刚刚运行 mm 的目录登录一个 `mm.config.json` 文件, 输入以下内容:

``` json
{
  "api": {
    "/start": {
      "message": "hello world"
    }
  }
}
```

提示: 这些更改不需要重新运行 mm 就能看到更改效果.

使用浏览器打开 http://localhost:9000/start , 你将会看到:
``` json
{
  "message": "hello world"
}
```

恭喜你, 实现了你的第一个 API .

再试试添加另一个 api 吧:
``` json {6-8}
{
  "api": {
    "/start": {
      "message": "hello world"
    },
    "/text": {
      "message": "some text"
    }
  }
}
```

::: details 可能遇到的问题
### 无法访问接口
- json 语法错误
- 访问的路径不是自己添加的路径 `/start` 或 `/text`
:::
