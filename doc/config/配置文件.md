# 配置文件
与 babel 或 eslint 这些工具一样, mm 支持 json, js 这些类型的配置文件.

### json
json 文件看起来简单, 但需要遵循严格的语法, 例如对象的键只能用引号包裹, 不能存在末键逗号.

**mm.config.json**
``` json
{
  "prot": 9000,
  "replayProt": 9001
}
```
但是, json 文件很容易支持跨语言解析.

### js
js 文件支持灵活的语法以及强大的编程功能.

**mm.config.js**
``` js
module.exports = {
  prot: 9000,
  replayProt: 9001,
}
```

作为函数使用的时候, 可以获取 mm 提供的[工具库](/config/config_fn.html).
**mm.config.js**
``` js
module.exports = util => {
  return {
    prot: 9000,
    replayProt: 9001,
  }
}
```
